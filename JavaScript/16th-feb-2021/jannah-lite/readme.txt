== Description ==

Jannah Lite is fast, fully customizable & beautiful WordPress theme suitable for blog, news and magazine website. It is very lightweight and offers unparalleled speed. It's a clean coded theme with seo freindly code structure.

It offers special features, one pre build homepage and great theme options through WordPress default customizer option and widgets. Works perfectly with all page builders like Elementor, Beaver Builder, Visual Composer, SiteOrigin, Divi, etc. Designed, Developed, Maintained & Supported by WPPool.

Checkout demo on: https://jannah.wppool.dev/

Looking for a perfect base theme? Look no further. Jannah Lite is fast, fully customizable theme that you can use for building any kind of website!

== Installation ==

= From WordPress Theme Directory =
1. Visit "Appearance > Themes > Add New"
2. Search for "Jannah Lite"
3. Install and activate

= From Downloaded File =
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
